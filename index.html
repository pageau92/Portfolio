About

I'm a front-end web developer specialized in HTML5, CSS, SASS JavaScript, and ReactJS Creating and deploying mobile-first user-friendly websites and applications became my passion when I was first introduced to programming in school in 2022. I began to teach myself how to code in JavaScript, as well as the fundamentals of web development while still attending school full-time and I became fascinated and curious about what else I can do with programming languages.
